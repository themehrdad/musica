# Musica — Piano Note Recognition Trainer

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** An iOS app where learners see random music notes on a staff, play them on a real piano, and get instant audio feedback on correctness — with profiles, daily goals, and progressive hints.

**Architecture:** SwiftUI app with SwiftData for local persistence (profiles + daily progress). AudioKit handles real-time microphone pitch detection. Custom SwiftUI Canvas draws the music staff and notes. Game state is managed via an ObservableObject coordinating the note display → listen → evaluate → animate → next-note loop.

**Tech Stack:** Swift 6.3, SwiftUI, SwiftData, AudioKit 5.x + SoundpipeAudioKit, PhotosUI, XcodeGen

                    .background(

                            startPoint: .topLeading,
                            endPoint: .bottomTrailing
                        )
                    )
            }
        }
        .frame(width: size, height: size)
        .clipShape(Circle())
    }

```swift
import Foundation

enum NoteName: String, CaseIterable {
    case C, D, E, F, G, A, B
}

struct MusicNote: Equatable {
    let name: NoteName
    let octave: Int        // 3–6
    let midiNumber: Int    // 48 (C3) – 79 (G5) per Config

    /// Semitones from C for each natural note
    private static let semitoneOffset: [NoteName: Int] = [
        .C: 0, .D: 2, .E: 4, .F: 5, .G: 7, .A: 9, .B: 11
    ]

    init(name: NoteName, octave: Int) {
        self.name = name
        self.octave = octave
        self.midiNumber = (octave + 1) * 12 + Self.semitoneOffset[name]!
    }

    init?(midiNumber: Int) {
        self.midiNumber = midiNumber
        let octave = midiNumber / 12 - 1
        let semitone = midiNumber % 12
        guard let match = Self.semitoneOffset.first(where: { $0.value == semitone }) else {
            return nil // It's a sharp/flat — skip if naturalNotesOnly
        }
        self.name = match.key
        self.octave = octave
    }

    /// Frequency in Hz (A4 = 440)
    var frequency: Double {
        440.0 * pow(2.0, Double(midiNumber - 69) / 12.0)
    }

    /// Closest MusicNote from a frequency. Returns nil if it's a sharp/flat and naturalNotesOnly.
    static func fromFrequency(_ freq: Double) -> MusicNote? {
        let midi = 69 + 12 * log2(freq / 440.0)
        let rounded = Int(round(midi))
        guard Config.noteRange.contains(rounded) else { return nil }
        return MusicNote(midiNumber: rounded)
    }

    /// Staff position: 0 = middle C (C4), each step = one line/space.
    /// Used by StaffView to place the note head vertically.
    /// Positive = above middle C, negative = below.
    var staffPosition: Int {
        let noteOrder: [NoteName] = [.C, .D, .E, .F, .G, .A, .B]
        let index = noteOrder.firstIndex(of: name)!
        return (octave - 4) * 7 + index
    }

    /// Generate a random natural note within Config.noteRange
    static func random() -> MusicNote {
        let allNotes = (Config.noteRange).compactMap { MusicNote(midiNumber: $0) }
        return allNotes.randomElement()!
    }

    /// Display name like "C4", "G5"
    var displayName: String {
        "\(name.rawValue)\(octave)"
    }
}
```

### Task 2.2: Staff Notation View

**Files:**
- Create: `Musica/Views/Practice/StaffView.swift`

- [ ] **Step 1: Implement StaffView using Canvas**

The staff view draws:
1. Five horizontal lines (E4, G4, B4, D5, F5)
2. A treble clef (𝄞 Unicode character)
3. A note head (filled ellipse) at the correct vertical position
4. Ledger lines for notes outside the staff (C4, D4, A5, B5, etc.)

Key geometry:
- `staffPosition 0` = C4 (middle C) = one ledger line below the bottom staff line
- Each staff position step = half the line spacing
- Staff line positions correspond to staffPositions: 2(E4), 4(G4), 6(B4), 8(D5), 10(F5)

```swift
import SwiftUI

struct StaffView: View {
    let note: MusicNote?
    let lineSpacing: CGFloat = 14
    let noteHeadWidth: CGFloat = 18
    let noteHeadHeight: CGFloat = 13

    // staffPosition for each of the 5 lines (bottom to top)
    private let staffLinePositions = [2, 4, 6, 8, 10] // E4, G4, B4, D5, F5

    var body: some View {
        Canvas { context, size in
            let staffWidth = size.width - 80 // leave room for clef
            let staffLeft: CGFloat = 70
            let midY = size.height / 2 // center of middle staff line (B4, position 6)

            // --- Draw 5 staff lines ---
            for (i, _) in staffLinePositions.enumerated() {
                let y = midY + CGFloat(2 - i) * lineSpacing // bottom line at +2, top at -2
                let path = Path { p in
                    p.move(to: CGPoint(x: staffLeft, y: y))
                    p.addLine(to: CGPoint(x: staffLeft + staffWidth, y: y))
                }
                context.stroke(path, with: .color(.primary.opacity(0.4)), lineWidth: 1.5)
            }

            // --- Draw treble clef ---
            let clefText = Text("𝄞")
                .font(.system(size: lineSpacing * 6.5))
                .foregroundColor(.primary.opacity(0.6))
            context.draw(clefText,
                         at: CGPoint(x: staffLeft + 20, y: midY + lineSpacing * 0.2))

            // --- Draw note ---
            if let note {
                let noteX = size.width / 2 + 20
                let noteY = yPosition(for: note.staffPosition, midY: midY)

                // Ledger lines
                let ledgerLines = ledgerLinesNeeded(for: note.staffPosition)
                for ledgerPos in ledgerLines {
                    let ly = yPosition(for: ledgerPos, midY: midY)
                    let path = Path { p in
                        p.move(to: CGPoint(x: noteX - noteHeadWidth - 4, y: ly))
                        p.addLine(to: CGPoint(x: noteX + noteHeadWidth + 4, y: ly))
                    }
                    context.stroke(path, with: .color(.primary.opacity(0.4)), lineWidth: 1.5)
                }

                // Note head (filled ellipse, slightly rotated)
                let noteRect = CGRect(
                    x: noteX - noteHeadWidth / 2,
                    y: noteY - noteHeadHeight / 2,
                    width: noteHeadWidth,
                    height: noteHeadHeight
                )
                let ellipse = Path(ellipseIn: noteRect)
                context.fill(ellipse, with: .color(.primary))
            }
        }
        .frame(height: lineSpacing * 12)
    }

    /// Convert a staffPosition to a Y coordinate
    private func yPosition(for staffPosition: Int, midY: CGFloat) -> CGFloat {
        // staffPosition 6 = B4 = center line
        // Each position step = half a lineSpacing
        return midY - CGFloat(staffPosition - 6) * (lineSpacing / 2)
    }

    /// Which staff-line positions need ledger lines for this note
    private func ledgerLinesNeeded(for staffPosition: Int) -> [Int] {
        var lines: [Int] = []
        if staffPosition <= 0 {
            // Below staff: ledger lines at 0 (C4), -2, -4... down to the note
            var pos = 0
            while pos >= staffPosition {
                lines.append(pos)
                pos -= 2
            }
        }
        if staffPosition >= 12 {
            // Above staff: ledger lines at 12 (A5), 14, 16... up to the note
            var pos = 12
            while pos <= staffPosition {
                lines.append(pos)
                pos += 2
            }
        }
        return lines
    }
}
```

### Task 3.1: AudioService — Microphone + Pitch Detection

**Files:**
- Create: `Musica/Services/AudioService.swift`

- [ ] **Step 1: Implement AudioService**

Wraps AudioKit's engine and PitchTap. Publishes detected note, amplitude, and whether the mic is active.

```swift
import Foundation
import AudioKit
import SoundpipeAudioKit
import Observation

@Observable
final class AudioService {
    var detectedNote: MusicNote?
    var amplitude: Float = 0
    var isListening = false

    private var engine = AudioEngine()
    private var pitchTap: PitchTap?
    private var mic: AudioEngine.InputNode?
    private var stableCount = 0
    private var lastMidi: Int?

    func start() throws {
        guard let input = engine.input else { return }
        mic = input

        // Silence the output so we don't hear feedback
        let silence = Fader(input, gain: 0)
        engine.output = silence

        pitchTap = PitchTap(input) { [weak self] pitchArray, ampArray in
            self?.processPitch(pitchArray[0], amplitude: ampArray[0])
        }

        try engine.start()
        pitchTap?.start()
        isListening = true
    }

    func stop() {
        pitchTap?.stop()
        engine.stop()
        isListening = false
        detectedNote = nil
    }

    private func processPitch(_ frequency: Float, amplitude amp: Float) {
        DispatchQueue.main.async { [weak self] in
            guard let self else { return }
            self.amplitude = amp

            // Ignore quiet sounds (voices, ambient noise)
            guard amp > Config.pitchAmplitudeThreshold else {
                self.stableCount = 0
                self.lastMidi = nil
                self.detectedNote = nil
                return
            }

            // Convert frequency to MIDI
            let midi = Int(round(69 + 12 * log2(Double(frequency) / 440.0)))

            // Require stability: same MIDI note for N consecutive frames
            if midi == self.lastMidi {
                self.stableCount += 1
            } else {
                self.stableCount = 1
                self.lastMidi = midi
            }

            if self.stableCount >= Config.pitchStabilityFrames {
                self.detectedNote = MusicNote(midiNumber: midi)
            }
        }
    }
}
```

Key design decisions for voice rejection:
- **Amplitude threshold** (`0.05`): Ignores quiet background chatter
- **Stability requirement** (`3 frames`): Piano notes produce stable pitch; speech slides around and rarely holds the same MIDI note for 3 consecutive analysis frames
- **Natural notes only filter**: `MusicNote(midiNumber:)` returns nil for sharps/flats when configured, adding another layer of noise rejection

### Task 3.2: PracticeViewModel — Game State Machine

**Files:**
- Create: `Musica/Services/PracticeViewModel.swift`

- [ ] **Step 1: Implement the game loop state machine**

States: `listening` → `correct` → `(animate)` → `listening` (new note)
                    → `wrong` → `(animate)` → `listening` (same note)

```swift
import Foundation
import SwiftData
import Observation

enum PracticeState: Equatable {
    case listening
    case correct
    case wrong
    case goalReached
}

@Observable
final class PracticeViewModel {
    var currentNote: MusicNote = MusicNote.random()
    var state: PracticeState = .listening
    var completedToday: Int = 0
    var wrongAttempts: Int = 0
    var showHint: Bool = false

    let profile: Profile
    let audioService = AudioService()

    private var modelContext: ModelContext?
    private var isProcessing = false

    init(profile: Profile) {
        self.profile = profile
    }

    func setup(context: ModelContext) {
        self.modelContext = context
        loadTodayProgress()
    }

    func startListening() throws {
        try audioService.start()
    }

    func stopListening() {
        audioService.stop()
    }

    /// Called when AudioService detects a stable note
    func evaluateNote(_ detected: MusicNote) {
        guard state == .listening, !isProcessing else { return }
        isProcessing = true

        if detected.midiNumber == currentNote.midiNumber {
            state = .correct
            completedToday += 1
            wrongAttempts = 0
            showHint = false
            saveTodayProgress()

            if completedToday == Config.dailyGoal {
                // Delay to show confetti first, then crown
                DispatchQueue.main.asyncAfter(deadline: .now() + 1.5) { [weak self] in
                    self?.state = .goalReached
                }
            } else {
                advanceAfterDelay()
            }
        } else {
            state = .wrong
            wrongAttempts += 1
            if wrongAttempts >= Config.wrongAttemptsBeforeHint {
                showHint = true
            }
            // Return to listening after animation
            DispatchQueue.main.asyncAfter(deadline: .now() + 1.5) { [weak self] in
                self?.state = .listening
                self?.isProcessing = false
            }
        }
    }

    func nextNote() {
        currentNote = MusicNote.random()
        state = .listening
        wrongAttempts = 0
        showHint = false
        isProcessing = false
    }

    func continueAfterGoal() {
        state = .listening
        isProcessing = false
        nextNote()
    }

    private func advanceAfterDelay() {
        DispatchQueue.main.asyncAfter(deadline: .now() + 1.5) { [weak self] in
            self?.nextNote()
        }
    }

    private func loadTodayProgress() {
        guard let context = modelContext else { return }
        let today = DailyProgress.todayString()
        let descriptor = FetchDescriptor<DailyProgress>(
            predicate: #Predicate { $0.date == today }
        )
        let results = (try? context.fetch(descriptor)) ?? []
        let match = results.first { $0.profile?.persistentModelID == profile.persistentModelID }
        completedToday = match?.notesCompleted ?? 0
    }

    private func saveTodayProgress() {
        guard let context = modelContext else { return }
        let today = DailyProgress.todayString()
        let descriptor = FetchDescriptor<DailyProgress>(
            predicate: #Predicate { $0.date == today }
        )
        let results = (try? context.fetch(descriptor)) ?? []
        if let existing = results.first(where: { $0.profile?.persistentModelID == profile.persistentModelID }) {
            existing.notesCompleted = completedToday
        } else {
            let progress = DailyProgress(date: today, notesCompleted: completedToday, profile: profile)
            context.insert(progress)
        }
        try? context.save()
    }
}
```

## Notes for Implementation

1. **AudioKit requires a real device** — the iOS Simulator has no microphone input. Phase 3 onward must be tested on a physical iPhone/iPad.

2. **Microphone permission** must be requested at runtime. The first call to `engine.start()` triggers the system permission dialog if `NSMicrophoneUsageDescription` is in Info.plist.

3. **Voice rejection strategy** is purely signal-processing-based (amplitude threshold + pitch stability). No ML needed. This works because:
   - Piano notes have very stable pitch; human speech constantly modulates
   - Piano has sharp attack and clear harmonics; speech is noisy
   - The amplitude threshold filters casual background conversation

4. **SwiftData daily reset** happens automatically — we query by today's date string, so yesterday's progress doesn't carry over.

5. **Natural notes only** (no sharps/flats) is appropriate for beginners and simplifies staff rendering (no accidental symbols needed). Configurable via `Config.naturalNotesOnly`.
